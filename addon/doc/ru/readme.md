# Введение

Это дополнение использует программу Scanvox для чтения ваших бумажных документов. Оно было создано с помощью разработчика программы и не требует дополнительной установки.

## Необходимые условия 

Для использования этого дополнения к компьютеру должен быть подключён USB-сканер, совместимый с TWAIN или WIA, что характерно для большинства сканеров.

## Как это работает

Чтобы использовать это дополнение, перейдите в меню NVDA и выберите Scanvox. В этом диалоге вы можете начать сканирование, нажав на кнопку Сканировать. Начало сканирования займёт несколько секунд, а по окончании процесса отсканированный текст будет автоматически прочитан вслух. Вы можете отключить автоматическое воспроизведение, перейдя в меню настроек NVDA, а затем в категорию scanvox для NVDA. Нажимайте эту кнопку до тех пор, пока не будут отсканированы все страницы. Если вы хотите удалить только что отсканированную страницу, нажмите кнопку «Удалить последнюю отсканированную страницу» , нажав tab после  кнопки «сканировать» или с помощью сочетания клавиш alt+l. После завершения сканирования вы можете прочитать отсканированные страницы, выполнив shift+tab от кнопки «сканировать» или alt+t для доступа к редактору с содержимым всех страниц,. Если вы хотите быстро переходить от страницы к странице, вы можете использовать кнопки «предыдущая страница» и «следующая страница» или сочетания клавиш ctrl+shift+стрелка вверх и ctrl+shift+стрелка вниз. Сохранить файл можно с помощью соответствующей кнопки.
Если вы хотите удалить отсканированные страницы, чтобы отсканировать новый документ, вы можете нажать кнопку "Удалить отсканированные страницы".
При выходе из Scanvox все отсканированные страницы будут удалены.

### Сочитания клавиш

Дополнение "Scanvox для NVDA" можно запустить из любого места на вашем компьютере, нажав nvda+alt+s. Этот жест можно изменить в диалоге  жесты ввода.
